# 🏦 Planejamento Previdenciário - Landing Page

Uma landing page interativa construída com Streamlit para incentivar e facilitar o planejamento de aposentadoria.

## 🚀 Funcionalidades

- **Landing Page Atrativa**: Seção hero, benefícios, estatísticas e call-to-action
- **Simulador Completo**: Cálculo detalhado de necessidade de poupança
- **Projeções Visuais**: Gráficos de crescimento patrimonial
- **Relatórios**: Download de relatórios em PDF
- **FAQ Interativa**: Perguntas frequentes com respostas
- **Formulário de Contato**: Captura de leads

## 📋 Pré-requisitos

- Python 3.8+
- pip
